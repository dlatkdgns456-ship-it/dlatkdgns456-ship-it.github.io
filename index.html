TOHOME

식품 쇼핑몰 클론 + AI 기능 연동

식품 쇼핑몰 화면을 구현한 뒤 FastAPI와 MySQL을 연결해 회원가입, 로그인, 상품 조회, 장바구니 같은 기본 기능을 만들었습니다. 상품 데이터를 활용하는 AI 챗봇과 이미지 분석 기반 상품 등록 기능을 추가해 프론트엔드, 백엔드, 데이터베이스, AI가 이어지는 흐름을 경험했습니다.

HEIGHTS. Store Clone

React 기반 패션 이커머스 웹 애플리케이션

React Router를 이용해 메인, 브랜드, 상품 상세, 장바구니, 고객센터가 이어지는 쇼핑몰 이동 흐름을 만들었습니다. Context API로 장바구니와 리뷰, 문의 상태를 여러 화면에서 공유하고 사용자의 조작이 바로 반영되도록 구현했습니다.

면접에서 설명할 핵심

  2. 상품 ID를 기준으로 상세 정보, 리뷰와 문의가 연결되도록 데이터를 구성했습니다.

BOOK ARCHIVE

Kakao Book Search API 기반 온라인 서점

Kakao Book Search API에서 실제 도서 정보를 받아 검색 결과와 상세 화면을 구성했습니다. 검색에서 상세, 장바구니, 마이룸으로 이동하는 흐름을 만들고 localStorage를 사용해 페이지가 바뀌어도 장바구니와 다크모드 상태가 유지되도록 구현했습니다.

형태개인 프로젝트

Kakao APIDynamic RenderinglocalStorageDark ModeEmpty State
면접에서 설명할 핵심

  2. 같은 bookCart 키를 사용해 메인, 상세, 장바구니, 마이룸의 상태를 연결했습니다.
